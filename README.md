# My Mastra App

このリポジトリは [Mastra](https://github.com/mastra-ai/mastra) を利用したサンプルアプリケーションです。

## 主要な機能

- **Weather Agent**: 指定した場所の天気情報を取得します。
- **Article Assistant**: Qiita、Zenn、note で公開した記事を基に新しい記事案を提案します。

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
